# 五子棋控制台版 - C++ OOP课程项目

## 项目简介
这是一个基于控制台的五子棋小游戏，使用现代C++特性编写，采用面向对象设计思想。

## 功能特点
- 15x15标准五子棋棋盘
- 支持黑白双方对战（X代表黑棋，O代表白棋）
- 开始可选择执黑或执白
- 输入xy坐标落子
- 自动判断五子连珠胜利
- 彩色控制台界面

## 操作说明
1. 启动游戏后选择：1-黑棋(X)先手，2-白棋(O)后手
2. 轮流输入坐标 x y（例如：7 7 表示第7行第7列）
3. 先形成五子连珠的一方获胜

## 编译与运行
### 环境要求
- 操作系统：Windows
- 编译器：MinGW GCC 6.3.0+ 或 Visual Studio 2017+
- C++标准：C++11 或更高版本

### 编译命令
g++ -std=c++11 -o gomoku.exe game.cpp

### 运行
.\gomoku.exe

## 项目结构
newgame/
- game.cpp          # 主游戏源代码
- gomoku.exe        # 编译后的可执行文件

## 类设计
- GameObject (基类): 抽象基类，定义游戏对象的通用接口
- ChessBoard (派生类): 15x15棋盘管理，落子与胜负判断
- Player (派生类): 玩家管理，输入处理
- Game: 游戏主管理类，控制游戏循环

## 现代C++特性使用
- std::vector 代替动态数组
- std::string 代替 char[]
- enum class 代替裸枚举
- auto 简化类型声明
- 范围 for 循环
- nullptr 代替 NULL
- 列表初始化
